# Structured Data Schema Map

## Global Configuration (`nuxt.config.ts`)

- **Organization**: Output automatically via `schemaOrg.identity` config.
  - `@id`: `<siteUrl>/#identity`
  - `name`: "Favicon Checker"
  - `url`: `https://favicon-checker.nard.uk`
  - `logo`: `<siteUrl>/favicon.svg`
- **WebSite**: Automatically scaffolded by generic template setup alongside
  identity.

## Route Maps

### `/` (Homepage)

- **WebPage**: Generated by `useAppSeo()` providing explicit absolute URL, Name,
  and Description.
- **FAQPage**: Generated by `useFAQSchema` injected inside `index.vue`. Includes
  5 structured answers for rich results in Google.

### `/batch` (Batch Check)

- **WebPage**: Generated by `useAppSeo()`, representing the utility layout
  explicitly linked to the tool's core entity.

## Assertion
